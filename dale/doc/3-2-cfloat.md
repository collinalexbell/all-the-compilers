## 3.2 cfloat

### Details

Module: cfloat

### Description

Provides the following floating-point macro constants:

  * FLT-RADIX
  * FLT-ROUNDS
  * FLT-DIG
  * FLT-EPSILON
  * FLT-MANT-DIG
  * FLT-MAX
  * FLT-MAX-EXP
  * FLT-MIN
  * FLT-MIN-EXP
  * DBL-DIG
  * DBL-EPSILON
  * DBL-MANT-DIG
  * DBL-MAX
  * DBL-MAX-EXP
  * DBL-MIN
  * DBL-MIN-EXP
  * LDBL-DIG
  * LDBL-EPSILON
  * LDBL-MANT-DIG
  * LDBL-MAX
  * LDBL-MAX-EXP
  * LDBL-MIN
  * LDBL-MIN-EXP
